Φ60×90L トースカン心出し後 作業手順
（STEP1〜STEP5）

STEP1：端面処理（基準点づくり）

■ 目的

ワークの端面を整え、Z軸の基準点（Z=0）を正確に作る。

■ 使用工具

  • 荒加工：外径荒バイト（超硬）
  • 仕上げ：外径仕上げバイト（サーメット）

■ 加工条件

  • 荒加工：回転数 570rpm、送りなし
  • 仕上げ：回転数 1800rpm、送りなし

■ 手順（荒加工）

  1. 刃先を端面に軽く触れさせる。→ 端面位置を確認するため。
  2. 中ハンドル（X軸）で刃物を外側へ戻す。→ 誤作業でミスを防ぐ等
  3. 小ハンドル（Z軸）をゼロセットし、0.4mm 切り込む位置まで進める。
  4. 中ハンドルを進めて端面を荒削りする。
  5. 削り終わったら中ハンドルで刃物を外へ戻す。

■ 手順（仕上げ）

  1. 刃先を端面に軽く触れさせる。
  2. 中ハンドルで外へ戻す。
  3. 小ハンドルをゼロセットし、0.1mm 切り込む位置まで進める。
  4. 中ハンドルを進めて端面を仕上げる。
  5. 削り終わったら中ハンドルで外へ戻す。

STEP2：外径荒加工（Φ60 → Φ40.5 → Φ30.5 → Φ22.5）

■ 目的

外径を段階的に荒加工し、仕上げに向けた寸法に近づける。

■ 使用工具

外径荒バイト（超硬）

■ 加工条件

  • 回転数 570rpm（途中で700rpmへ変更）
  • 送り：0.3mm/rev（6E）

■ 手順（基準点づくり・ケガキ）

  1. 刃先を端面に軽く触れさせる。→ Z軸の基準点を確認。
  2. 中ハンドルで外へ戻す。
  3. 大・小ハンドル（Z軸）をゼロセットし、Z=0 を確定する。
  4. 大ハンドルを左へ進め、60.0mm の位置に合わせる。
  5. 中ハンドルを進めて刃先を軽く当て、ケガキ線（目印）を入れる。
  6. 中ハンドルを戻し、大ハンドルで刃先を基準点より右へ戻す。

■ 手順（Φ50 → Φ40.5 / Φ40.5 → Φ30.5 / Φ30.5 → Φ22.5）

※ 基本流れは共通（ケガキ位置と回転数変更のみ異なる）

  1. 中ハンドルで長手方向に刃先を軽く触れさせる。
  2. 大ハンドルで刃先を基準点より右へ戻す。
  3. 中ハンドルをゼロセットし、5.0mm 切り込む位置まで進める。
  4. 自動送りでケガキ付近まで削る。
  5. 送り停止後、大ハンドルを左に進めケガキ位置まで合わせる。
  6. 大ハンドルで刃先を右へ戻す。
  7. ノギス測定 → Φ目標寸法になるよう残りを追加切込で削る。
ケガキ位置：Φ40.5は59.9mm / Φ30.5は44.9mm / Φ22.5は24.9mm（回転数700rpmに変更）

STEP3：溝加工（Φ22.5 → Φ18.0）

■ 目的

幅4mmの溝バイトで、Φ18.0 の溝を加工する。

■ 使用工具

溝バイト（HSS）

■ 加工条件

  • 回転数 360rpm
  • 送りなし（手送り）

■ 手順

  1. Φ30.5 の端面に溝バイトの左側を軽く触れさせる。
  2. 中ハンドルを進め、Φ22.5 に軽く触れるまで削る。
  3. 中ハンドルをゼロセットし、2.0mm 切り込む位置まで進めて削る。
  4. 削り終わったら中ハンドルで外へ戻す。
  5. ノギス測定 → Φ18.0 になるよう残りを追加しながら削る（達成したら中ハンドル記録）。
  6. 溝幅5mm確保のため、大ハンドルを右へ1mm進めて記録位置まで追加削り。

STEP4：外径仕上げ（Φ22.5 → Φ21.75 / Φ30.5 → Φ30.0）

■ 目的

荒加工で残した外径を、仕上げ寸法まで整える。

■ 使用工具

外径仕上げバイト（サーメット）

■ 加工条件

  • 回転数 1800rpm
  • 送り：0.1mm/rev（1F）

■ 手順（Φ22.5 → Φ21.75）

  1. 刃先を長手方向に軽く触れさせる。
  2. 大ハンドルで刃先を基準点より右へ戻す。
  3. 中ハンドルをゼロセットし、0.25mm 切り込む位置まで進める。
  4. 自動送りで仕上げ切削を行う。
  5. マイクロ測定 → Φ21.75 になるよう残りを追加して削る。

■ 手順（Φ30.5 の端面整え / Φ30.5 → Φ30.0）

端面さらった後、同様に仕上げ加工（詳細は清書版通り）

STEP5：面取り（C面加工）

■ 目的

外径の角部を適切な大きさで面取りし、仕上げ品質を整える。

■ 使用工具

面取りバイト（HSS）

■ 加工条件

  • 回転数 142rpm
  • 送りなし

■ 手順

  • M22（Φ21.75）：油塗布 → C2（中ハンドル4mm or 小ハンドル2mm）
  • Φ30：油塗布 → C1（中ハンドル2mm or 小ハンドル1mm）
  • Φ40：油塗布 → C1（中ハンドル2mm or 小ハンドル1mm）
